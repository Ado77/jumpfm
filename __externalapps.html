External Programs

You can bind hotkeys to trigger the execution of external applications. You can pass those applications arguments based on the current state.

Available Variables:

Variable Description
{cur0} The file under cursor on the left panel
{cur1} The file under cursor on the right panel
{pwd0} The full path of the location of the left panel
{pwd1} The full path of the location of the right panel
{active} The full path of the active panel
{selected} The selected files

Examples:

git difftool {selected} will trigger the diff tool configured for git with the selected files passed as arguments to the program.

git difftool {active} will trigger the diff tool configured for git with the full path of the active panel passed as an argument to the program.
