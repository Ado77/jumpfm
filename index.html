About

JumpFm is a minimalistic dual panel file manager for Linux that lets you navigate your file system very efficiently.

Jumping

Press j to jump directly to your favorites folders. No manual bookmarking is needed, JumpFm will learn where you want to go.

Git Awareness

Files are displayed according to thier git status.

Filtering

Press f to quickly find the files you are looking for.

Flat Mode

Press r and all directories are gone. Press it again to bring them back.

Instant Gist

Press ctrl + g to create instant gist from selected files.

Before You Jump

Instant gists are public !!! don't gist your private ssh key or any other private information.

There are no confirmation dialogs in JumpFm, whenever you hit the DEL key a file is gone.

Key Bindings

You can change tde default keybindings by editing tde keybinding file. Not all keybindings are available (from obvious reasons) when the filter box is active.

Plugins

Plugins control every aspect of JumpFm. You can enable/disable them, install new once and also write your own.

Builtin plugins:

Plugin Enabled Description Safely Disable
clock true Displays the date and time. yes
version true Displays the current version of JumpFm. yes
file system true List the files in the directory. no
key navigation true Enable keyboard shortcuts. no
file operations true Enable file operations such as renaming, editing, etc... no
copy true Enable file copying. no
jump true Enable directory jumping. yes (but why?)
history true Enable history navigation. yes (but why?)
flat mode true Enable flat mode. yes
zip true Enable zipping and unzipping files. yes
git status false Enable git status indicators. Enabling this plugin might cause some performance degradation when browsing large git repositories. yes
gist false Allow sharing files via public gists. If you decide to enable this plugin make sure you know what public means. yes
