Default Key Bindings

You can change tde default keybindings by editing tde keybinding file. Not all keybindings are available (from obvious reasons) when tde filter box is active.

navigation
j open tde jump dialog
down down
shift + down down and select
page down page down
shift + page down page down and select
up up
shift + up up and select
page up page up
shift + page up page up and select
home go to start
shift + home go to start and select
end go to end
shift + end go to end and select
enter enter folder / open file
backspace up tde directory tree
ctrl + home go to home directory
ctrl + right open directory in right panel
ctrl + left open directory in left panel
alt + left go to previous directory
alt + right go to next directory
selection
space toggle selection
ctrl + a select all
esc clear selection
file operations
del delete selected files
F4 edit file
Shift + F4 new file
F7 new folder
F2 rename file
z zip selected files
u unzip current file
p copy full patd of current file to clipboard
F5 copy selected files
F6 move selected files
panels
tab switch panel
s swap panels
filters
f open tde filter box
l show files witd same extension
h show/hide hidden files
r toggle flat mode
miscellaneous
ctrl + g create a public gist from selected files
ctrl + = increase font size
ctrl + - decrease font size
ctrl + 0 reset font size
